1. How to Work With “FIXPAPA”as a Engineer

  1. (a) To become anengineer with“FIXPAPA”You need to contact with registered vendor or partner at “FIXPAPA”. The registered vendor will authorize you as an engineer. As per your technical qualifications.
  2. (b) Once vendor register you as aregistered engineer. You can log in through web & app, further can accept or schedule the calls.
  3. (c) Now You can access engineer panel you can see 1st option.
    1. Open Jobs: - - In this open jobs option you can see you’re all jobs like Today, tomorrow and all jobs.
    2. By using “Today” job option you can see all the current jobs which you must attend today according to time scheduled.
    3. By using “Tomorrow” job option you can see all the tomorrow jobs which will you must attend according to time scheduled.
    4. By using “All jobs” option you can see all day after tomorrow jobs which will must attend according to time scheduled.
  4. (d) In Progress Jobs: - into this option you can see two options.
    1. On Going :- In this option you can see all your in progress jobs which you attend on your scheduled time.
    2. Completed: -In this option you can see your completed jobs.
  5. (e) My Info: -Into this option you can see your all the added Information.
